### Pre-Requisties
- Mock Api to run the app

### App configuration
- create `.env` and put Api base url and Google map api key

### Steps to run the app

Requires Node.js ver. 4+

1. Install dependencies:
```
npm install
```

2. Run test cases:
```
npm test
```

3. Run the server:
```
npm start
```

### Create production build

Requires Node.js ver. 4+

1. Install dependencies:
```
npm install
```

2. Create production build:
```
npm run build
```
